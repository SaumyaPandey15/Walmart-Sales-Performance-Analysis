# Walmart Sales Performance Analysis

## Project Overview

This project demonstrates an end-to-end retail sales analysis workflow using Python, SQL, and Power BI. It covers data cleaning, exploratory data analysis (EDA), SQL-based data querying, and business intelligence dashboarding to generate actionable insights from Walmart sales data.

---

## Project Workflow

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Integration
- Power BI Dashboard
